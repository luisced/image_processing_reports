# Listas

Una lista es una colección ordeanada de cualquier tipo de datos, nos permite manipularla y trabajar con sus datos.

Para definir una lista utilizaremos "[]" y dentro de ellos pondremos los valores que queremos almacenar ahí.

```py
lista = [1, 2, True, "Hola", [1, 2, 3]]
```

##Índices

Las listas, al ser conjuntos ordenados de datos, están indexadas, esto quiere decir que cada dato dentro de la lista, tiene un índice específico, estos índices empiezan desde cero, por lo que si una lista tiene tres elementos, estos se verán de la siguiente manera:

```py
lista = [1, 2, 3]

print(lista[0])
print(lista[1])
print(lista[2])
```
Otuput de la consola

```py
>>> 1
>>> 2
>>> 3
```
Utilizamos "[]" para accesar a los valores almacenados en ese índice de la listas

Podemos encontrar el último valor de la lista utiliando índices negativos de la siguiente manera:

```py
print(lista[-1])
```

Output de la consola

```py
>>> 3
```
## Slicing

El slicing es un procedimiento que nos ayuda a seleccionar una sección específica de la lista aprovechando que esta es indexada, por lo que utilizaremos los índices para especificar la sección que queremos seleccionar de la siguiente manera:

```py
lista = [1, 2, 3]

print(lista[0:2]) # aquí estamos indicando que vamos a cortar 
# desde el inicio hasta el segundo valor
```
Output de la consola
```py
>>>[1, 2]
```

Otro ejemplo de esto:
Podemos dejar el espacio vacío y esto va a indicar que vamos a ir desde el inicio o hasta el final respectivamente

```py
print(lista[:2]) # del inicio hasta el índice 2

print(lista[2:]) # del índice dos hasta el final
```
Otuput de la consola

```py
>>> [1, 2]

>>> [3]
```

## Funcion len

Podemos utilizar la función "len" para conocer la longitud de nuestra lista, es decir la cantidad de datos almacenados en nuestra lista, la usaremos de la siguiente manera:

```py
print(len(lista))
```
Output de la consola
```py
>>> 5
```
