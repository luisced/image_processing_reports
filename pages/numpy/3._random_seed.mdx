# Funciones NumPy

## Random 
El módulo `numpy.random` implementa generadores de números pseudoaleatorios con la capacidad de extraer muestras de una variedad de distribuciones de probabilidad. En general, los usuarios crearán una instancia de Generador con default_rng y llamarán a varios métodos para obtener muestras de diferentes distribuciones.

```py 
import numpy as np
np.random.random(4) #Usamos la función random, cabe aclarar que random() lo único que genera son floats aleatorios entre el 0.0-1.0
```   

Output:

```py
>>> array([0.37072208, 0.92212866, 0.84127822, 0.65552592])
```

## Seed 

La seed es el generador de números pseudoaleatorios, la seed va cambiando dependiendo de diferentes factores pero pdoemos modificar de cierta manera que valores nos salen cuando nosotros introducimos una seed especifica.

```py
import numpy as np
np.random.seed(123) #declaramos nuestra propia seed 123
print (np.random.rand(4)) #nos daran 4 valores diferentes en una lista en un rango de 0.0-1.0
```

Aclaración! Si quieres que te den los mismos números cada vez tienes que ejecutarn antes `np.random.seed(123)`

Output:
```py
>>> array([0.69646919, 0.28613933, 0.22685145, 0.55131477])
```
