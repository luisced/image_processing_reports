### Funcion de apilamiento de numpy

La funcion hstack() de numpy nos permite colocar dos o más plots de manera secuencial para ser capaces de tener una mejor visibilidad al momento de analizar los cambios realizados entre imágenes.

La función se usa de la siguiente manera:

```py
np.hstack((iamgen1, imagen2))
```
Para poder imprimirlo usamos plot como es costumbre:

```py
plt.plot(np.hstack((iamgen1, imagen2)))
```
