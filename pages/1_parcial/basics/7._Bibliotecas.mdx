# Bibliotecas

Cuando hablamos de bibliotecas en Python nos refereimos a un conjunto de módulos o funciones predefinidas que van más allá de las funciones del lenguaje lo que hace que algunas tareas se puedan facilitar.
Existen bibliotecas para muchas necesidades y con muchas nuevas funcionalidades, como lenguaje matemático, generación de gráficas, modificación de archivos, bases de datos y muchos ejemplos más.

Para el ámbito del procesamiento digital de imágenes, las bibliotecas que vamos a utilizar más serán:

- **Numpy**
- **Open CV**
- **Matplotlib**

Para importar cualquier biblioteca se hace de la siguiente manera, utilizaremos como ejemplo a Numpy

```py
import Numpy
```

También, podemos hacer uso de un alias al momento de importar una biblioteca, esto es para que sea más fácil sintácticamente al momento de usarla en el código

```py
import numpy as np
```
