## ¿Qué es OpenCV?

OpenCV (Open Computer Vision Library) es una biblioteca de software de código abierto que proporciona herramientas optimizadas para visión por computadora en tiempo real. Además, ofrece soporte para la ejecución de modelos de aprendizaje automático (Machine Learning) e inteligencia artificial (AI). La biblioteca es altamente optimizada y se centra en aplicaciones en tiempo real. Ofrece interfaces para varios lenguajes de programación como C++, Python y Java, y es compatible con múltiples plataformas como Linux, MacOS, Windows, iOS y Android.

## Características Principales

- **Código Abierto**: OpenCV es de código abierto y se distribuye bajo la licencia Apache 2. Esto significa que es gratuito para uso comercial.
- **Optimizado para Tiempo Real**: La biblioteca está diseñada para ser altamente eficiente, con un enfoque en aplicaciones en tiempo real.
- **Multiplataforma**: Ofrece soporte para múltiples lenguajes de programación y sistemas operativos, lo que lo hace versátil y ampliamente utilizable.

[Fuente: Página oficial de OpenCV](https://opencv.org/)
