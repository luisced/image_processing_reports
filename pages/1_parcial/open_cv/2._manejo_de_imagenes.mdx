## Acceder a un pixel en particular

```python
import cv2

image = cv2.imread("image.jpg")    # Read the image

# Access a specific pixel
x = 100    # x-coordinate
y = 200    # y-coordinate

pixel_value = image[y, x]    # Access the pixel value at (x, y)

print(pixel_value)
```
